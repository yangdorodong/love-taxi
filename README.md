# 🚕 Love Taxi — 커플 택시 미터기

데이트할 때 폰 하나로 미터기를 켜두면, 택시 요금처럼 **뽀뽀·안마·소원권**이 쌓이는 커플용 웹앱.

## 사용법
1. 폰 브라우저로 접속 → **탑승!** 버튼
2. 함께 있는 동안 시간과 이동 거리에 따라 보상이 적립돼요
3. 헤어질 때 버튼을 **길게 눌러** 운행 종료 → 영수증 📸

## 특징
- **병산제**: 한국 택시처럼 저속·정차 땐 시간요금, 이동 중(15.7km/h↑)엔 거리요금(GPS)
- **커스텀 요금표**: "10분당 뽀뽀 1개" 식으로 원하는 보상과 단가를 직접 설정
- **할증**: 심야(22~04시)·주말 자동 할증, 기념일·기분 할증
- **영수증 & 기록**: 운행이 끝나면 캡처하고 싶은 영수증, 지난 데이트 기록 보관
- 서버 없음 — 모든 데이터는 내 폰(localStorage)에만 저장돼요

## 개발
정적 웹앱 (바닐라 JS + ES 모듈, 빌드 없음).

```
python -m http.server 8347   # 로컬 실행
```

- `?mockgps=1` — 가짜 GPS (정지 30초 ↔ 30km/h 60초 반복)
- `?debug=1` — 디버그 패널 (fix 필터링 로그, 미터 상태)

GPS는 HTTPS(또는 localhost)에서만 동작해요.
